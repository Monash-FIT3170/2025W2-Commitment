# Git Standards

#### Branches

**Master**
Please never commit direct to master. Make changes through a pull request. Squash when merging.

**Other people's branches**
Please don't make commits to branches owned by other people without first asking the owner. If you must, make changes on your own branch, then raise a PR into the other person's branch, so they can review your changes, and we avoid conflicts

#### Branch naming convention
Please prefix your branch name with the type of branch it is.
- `feat/` for features
- `fix/` for any bug fixes or otherwise
- `temp/` for any garbage temporary branches
